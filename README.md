Features

- Add, edit, and delete tasks
- Add and update deadlines for tasks
- Dashboard with completed and pending task
- User Authentication
- Progress Bar to track productivity
- Personalized tasks for each user

Tech Stack

Frontend:
  - React (Vite)
  - CSS / Bootstrap

Backend:
  - Node.js
  - Express.js

Database:
  - MongoDB (Mongoose)

Authentication:
  - JWT (JSON Web Token)
